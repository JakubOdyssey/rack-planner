# Warehouse Rack Planner

A smart pallet placement planning tool designed for warehouse environments.

## Features

- Automatic pallet placement optimization
- Mixed pallet support
- Double stacking logic
- Partial rack usage (custom bay selection)
- Visual rack layout
- Export to operator-friendly PDF

## Use Case

This tool helps:
- optimize rack usage
- reduce manual decision-making
- improve warehouse efficiency
